# 🎓 中学受験サポートアプリ

子どもの中学受験のために、志望校の情報を夫婦で共有・比較するためのスマホ向けWebアプリです。

## 📱 公開URL（スマホで開けます）

**https://iizuka1478.github.io/-/**

- 上のURLをスマホのブラウザで開くと、そのまま使えます（夫婦どちらの端末でもOK）。
- iPhoneは「共有」→「ホーム画面に追加」、Androidは「メニュー」→「ホーム画面に追加」でアプリのように使えます。
- データを更新して `main` ブランチへ反映すると、サイトに自動で反映されます（数分以内）。


## できること

- 🏫 **学校一覧** … 偏差値・距離・受験日・募集人数で並べ替え
- 📍 **距離表示** … 自宅の最寄り駅を基準に、各校までの距離を自動計算＋ルート検索リンク
- 📊 **一覧の可視化** … 難易度・偏差値・募集人数・受験日・科目を一覧／詳細で確認
- ✨ **特徴・特色** … 各校の詳細ページに表示
- 📅 **カレンダー** … 説明会・文化祭・出願・受験などの予定を日付順に表示

## 使い方

特別な準備は不要です。`index.html` をブラウザで開くだけで動きます（スマホでもPCでも可）。

1. `index.html` をブラウザで開く
2. 「⚙️ 設定」で自宅の最寄り駅（駅名・緯度・経度）を登録
3. 「🏫 学校一覧」「📅 カレンダー」を閲覧

> スマホのホーム画面に追加すると、アプリのように使えます。

## 学校データの追加・編集

すべての学校情報は **`js/data.js`** にまとめてあります。
ファイル冒頭のコメントに各項目の意味を書いていますので、そこに追記・編集してください。
（現在は動作確認用にサンプル校が入っています。志望校のリンクをもとに実データへ差し替えます）

## 構成

```
index.html      画面の骨組み
css/style.css   見た目
js/data.js      学校データ（ここを編集）
js/app.js       表示ロジック
```

バックエンド不要・完全オフライン動作。設定（自宅最寄り駅）は端末のブラウザに保存されます。
